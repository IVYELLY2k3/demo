# Child Tracing System (Frontend + Backend)

## Setup and Run

1.  **Install Requirements:**
    Open your terminal in this folder and run:
    ```powershell
    py -m pip install -r requirements.txt
    ```

2.  **Run the App:**
    ```powershell
    py app.py
    ```

3.  **Access:**
    Open your browser and search for: `http://127.0.0.1:5000`

### Note on DeepFace
The first time you run `py app.py` or install dependencies, it might download large model weights (VGG-Face). This is normal and only happens once.
